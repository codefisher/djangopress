Toolbar Buttons

Toolbar Buttons is the single biggest button pack for Firefox or Thunderbird there is. There are now nearly 150 Firefox and 100 for Thunderbird. You will find buttons that enable hidden functionality or is too many clicks away, but some of the best buttons are those unique to the extension. For example from some of out most popular buttons, there are menu items for, Options, Bookmark Manager, Bookmark Page, and Add-ons, but for Snap Back, Restart and about:config there is no existing user interface. There have been nearly 2 million downloads to date.

To find out more information about each of the buttons, please see our Button List which has descriptions of all the buttons.

Custom Button Maker

If you don't want to have all the button in Toolbar Buttons, but rather just the sub set that you want, you can use the Custom Button Maker to create just that. It allows you to select buttons from a list and have an extension made with just your selection. It also has a number of extra features over the main release, including, extra buttons, expanded options window.

The Custom Button Maker also servers as a testing ground for new buttons and features, those that prove popular are added to the main extension.

Link Buttons

We have been asked for buttons that open many different web pages, so we created a simple tool, the link button maker, so that you can easily create toolbar buttons to any page you wish.

Colour Picker

Sharp Colour Picker a tool that lets you pick colours using colour wheel, web safe colour pallet, and HSL Panel.

MinimizeToTray Plus

MinimizeToTray Plus will minimise the current window to the system tray for Firefox, Thunerbird and Sunbird among others. It also has a selection of options including ones to enable it to minimize the window instead of closing and open the application at system start-up.

Pastel SVG Icons

Pastel SVG is an icon set based on the popular silk icons found as FamFamFam.com. It uses the same style, so some of the icons are almost identical. The goal of the project is to recreate the entire set in a full range of sizes while also adding many icons that we think the original set is missing.
