Translation, Please Help!

This has been the first update in a long time, and I have had problems getting on contact with the members of the translation team, so many of the translations in this release are incomplete. Could you help out? Can you be that one in 10,000 person that needs to respond to have all the translations finished? If you can, please head over to BabelZilla and sign up to extensions translation team!

Custom Button Maker

In addition to the above, there has been a new release of the Custom Button Maker. It has all buttons and features as the offical release along with the following extra features.
  • As suggested by the name, it will only include the buttons you want, and for the applications that you want to use it in.
  • The option of generating toolbars to hold the buttons that you want.
  • The option of using larger icon sizes.
  • An optional experimental feature that creates a menu to access all the buttons.
  • A few extra buttons that are were not thought to be popular enough for the main release.

There is a new simplified bug tracker for asking for, and then voteing on requests for new buttons. So if you would like something else to be added to the extension please make a request. If it then proves popular once released onto the Custom Button Maker, it will be added to the next release offical. If not, it will always remain avaliable there.

Aims for the next release

The next release is not intened to make any major changes, what we have in mind is.

  • Working towards having the translations updated.
  • Re-touching some of larger 24x24 icons.
  • Fixing any bugs found.
